# Victim Lab Script - by kader11000

## How to Use:

1. Make the script executable:
   ```bash
   chmod +x victim_lab.sh
   ```

2. Run the script:
   ```bash
   ./victim_lab.sh
   ```

## Requirements:
- bash
- zenity
- msfvenom
- msfconsole
- adb
- notify-send
- wget, tar, make, mail, scp

Make sure to edit the script to add your email, remote server info, and other personal settings.
